# 🔣 알고리즘 : Floyd Warshall

정점과 정점 사이의 최단 거리를 구할 때 사용할 수 있는 알고리즘  
  
정점과 정점을 잇는 엣지 사이에 거쳐갈 노드를 1~N 까지 하나씩 늘려가면서 (1개 2개 3개 ... N 개)  
  
거쳐갈 노드가 한개씩 추가될 때 마다 새로 노드와 노드 사이의 거리를 계산해준다  
  
거쳐갈 노드를 하나씩 추가하기 때문에 이번 회차에서 구한 거리는 정점이 k 개 있을 때 갈 수 있는 최단거리를 다 구한다

## ❗ 알고리즘

이렇게 하면 모든 케이스가 다 커버가 되나 싶지만, 한 회차씩 생각해보면 거쳐갈 수 있는 노드집합 대해서 모든 최소 거리를  
  
다 구한셈이기 때문에 k 개의 집합에 대해서도 성립한다
