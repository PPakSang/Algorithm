# 🔣 양궁대회

완전 탐색을 통해 조건에 부합하는 배열을 초기화해서 최종 결과를 반환 

## ☑️BackTracking

라이언이 과녁에 맞추는 화살을 모두 탐색하기 위해서 BackTracking 을 사용하였다<br>
반복문과 재귀문 종료조건을 합치면 배열 백트래킹이 완성되고, 종료조건 부합할 때 동작을 잘 설정하는 것이 핵심<br>

## ☑️중복조합

이 문제를 완전탐색으로 풀기 위해서는 여러가지 방법이 있을 수 있는데, 내가 이 문제에서 채택한 방식은<br>
중복조합을 통한 찐 완전 탐색(10개의 화살을 11개의 과녁에 꽂는 경우)이다.<br>
11H10 = 20C10 = 184756,, 완전탐색이 되구나 라고 생각할 수 있어야한다.
