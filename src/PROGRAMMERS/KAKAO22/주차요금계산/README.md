# 🔣 주차요금계산

신고결과받기와 유사한 문항<br>
문자열 파싱 후에 데이터를 어떤 방식으로 저장할지 고민이 필요한 문제<br>
저장해야할 데이터는 시간 - 차량번호 - 출입여부 의 형태

## ☑️Map<Key, Value> = new HashMap<>();

Key - Value 구조로 이루어진 Map을 이용해서 데이터 구조를 만들었다<br>
차번호 - Integer[최근 출입 시간, 누적 사용 시간, 출입상태, 요금]<br>
나갈 때에만 누적 시간을 체크하면 되므로 출입 시간만 알고있으면 된다.<br><br>

Map.getOrDefault 를 다음에 사용해보자

## ☑️Logic
차량이 나갈 때 누적 주차장 사용 시간을 업데이트 해주고<br>
하루가 끝났을 때 차량들을 살펴보면서 아직 출차하지 않은 차량에 대해서는 23:59분을 기준으로 출차 요금을 매긴다
