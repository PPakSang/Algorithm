## ☑️ 풀이

### 그래프 탐색에서 이전에 탐색 정보 남기기

visited 배열만 이용해서 그전에 해당 칸에 방문했을 때 정보를 남기면 된다.  
  
이 문제에서는 자동차가 임의의 칸에 도착했을 당시의 cost 를 visited 에 기록하여야 했는데  
  
다음번에 방문할 때에는 그 전에 자동차가 방문했을 때의 비용대비 현재의 비용을 비교하여  
  
탐색을 계속 진행할지 말지 결정할 수 있음.  
  
bfs 로도 할 수 있는데 dfs 에서 재귀를 생성하는게 bfs 에서는 q 에 넣는 것으로 생각하면 된다.  
  
즉 특정 조건으로 완전 탐색 해야할 때는 dfs bfs 둘 다 구현할 수 있다.
