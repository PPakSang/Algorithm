### 비트마스킹

nC1 .. nC2 .. nCn 까지의 조합이 모두 필요할 때 쓸 수 있는 방법  
  
그 중에서도 테이블과 column 에 따른 row 를 모두 선택해야할 때 쓸 수 있다.  
  
아이디어는 col 이 4개있는 테이블에서 가령 index 1번, 3번 col 을 고르고 싶다면 
for (i~n) -> arr[1][i] arr[3][i] 로 해도 괜찮지만  
  
0101 이라는 마스크를 설정하고 1 << colNum 으로 비트를 설정하고 0101 & 1 << colNum 을 했을 때 0보다 크다면  
해당 col 은 내가 선택하려는 col 이 맞을 것이다.
  
record 를 탐색할 때는 row 별로 탐색을 해나가는 방식을 전개하자
